# Project-2-Business-Analysis

📊 Project Overview
End-to-end business analysis project using Medallion Architecture for data cleaning, transformation, and analysis.

🏗️ Architecture:
# Medallion Layers:
- **Bronze**: Raw/ingested data (minimal transformation)
- **Silver**: Cleaned, validated, deduplicated data
- **Gold**: Business-ready aggregated data for BI tools

📈 Key Performance Indicators Tracked:
| Metric | Value |
|--------|-------|
| Total Sales | $206M |
| Total Profit | $77M |
| Profit Margin | 37% |
| Revenue YoY Growth | 20.36% |
| Total Quantity | 90K units |

🎯 Business Insights::

1. Strong Performance:
- Healthy 37% profit margin
- Impressive 20.36% revenue growth year-over-year
- Effective cost management relative to sales growth

2. Areas for Improvement:
1. **COGS Optimization**: Manufacturing cost ($129M) exceeds profit ($77M)
2. **Product Line Review**: Consider discontinuing low-profit products
   - HL Road Frame
   - Long-Sleeve Logo Jersey
   - LL Road Frame
   - Chaining

3. Seasonal Strategy:
- Peak months: May, June, December
- Target cities: Montreal, Alberta, Bengaluru, Bhubaneswar
- Focus on cycling benefits in dense cities (Bengaluru)

4. 🔧 Tools Used:
- SQL (Data Cleaning)
- Power BI / Tableau (Dashboard)
- DAX (Measures & Calculations)

 📌 Recommendations
1. Negotiate with suppliers for better pricing
2. Target seasonal promotions in low-sales months
3. Strengthen online and stockist channels
4. Partner with Amazon, Decathlon for online sales
